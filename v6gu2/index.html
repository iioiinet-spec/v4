4 Beat Stroke Engine

1. 4비트 스트로크 물리 알고리즘

4비트 스트로크(4 Beat Stroke)는 한 마디에 4분음표 4개가 균일하게 정박으로 떨어지는 가장 기본적이고 대중적인 음악 리듬 형태입니다.

4비트 스트로크 리듬 타임라인 구성: 1박자: 다운 스트로크 (Down Strum) 2박자: 다운 스트로크 (Down Strum) 3박자: 다운 스트로크 (Down Strum) 4박자: 다운 스트로크 (Down Strum)

매 박자 타이밍에 맞춰 피크가 6줄을 순차적으로 타격하는 다운 스트로크 사운드 믹서가 가동됩니다.

2. BPM 메트로놈 루프 스케줄러

브라우저 환경에서 밀림 없는 정밀한 메트로놈 템포 박자를 구현하기 위해 분당 비트 수(BPM) 기준 밀리초 스케줄러 회로를 장착했습니다.

박자 주기 시간(ms) 계산 공식: Interval = (60 / BPM) * 1000 BPM 95 설정 시: (60 / 95) * 1000 = 약 631.57ms 마다 1비트 연주

한 번 코드를 선택하면 Web API 타이머가 연주 상태 변수를 감시하여 정해진 주기 주기마다 자동으로 6줄의 아르페지오를 소환합니다.

3. 박자 동기화 음파 간섭 시각화

4비트 자동 메트로놈 타이밍에 완벽하게 락업(Lock-up)되어 6개의 음파 링이 주기적으로 뿜어져 나옵니다.

사용자가 화면의 빈 공간을 터치하거나 마우스를 휘저으면, 자동 연주되고 있는 4비트 주파수 음파 링들과 동적 마우스 기류가 유기적으로 교차 및 간섭을 일으키며 환상적인 물리적 잔상 무늬를 빚어냅니다.
