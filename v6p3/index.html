3옥타브 모바일 음파 실험실

키보드 자판 확장 및 하이브리드 멀티터치 시스템 인터랙션

현재 구역: 4옥타브 (가온음역)

3-Octave Mobile System

1. 3옥타브 광대역 주파수 연산

가온다역(4옥타브)을 기점으로 총 3개 옥타브(24개 음계)의 수치 해석 공식을 포함합니다.

주파수 대역 연산 범위: 최저음(C4): 261.63 Hz 최고음(B6): 1975.53 Hz 옥타브 공식: Freq * 2 = 1옥타브 상승

고음 대역으로 도약할수록 물결 벡터장의 맥동 속도 계수가 수학적으로 증폭되도록 가중치를 제어했습니다.

2. 모바일 하이브리드 멀티터치

스마트폰 환경에서의 부드러운 다중 연주(화음)를 지원하기 위해 모바일 전용 이벤트 핸들러를 구축했습니다.

  • touch-action none: 모바일 브라우저 고유의 화면 스크롤 및 더블클릭 화면 확대를 무효화하여 터치 렉을 완전히 제거합니다.
  • 정적 다중 포인터 추적: 여러 손가락이 동시에 다른 건반 영역을 연타해도 음향 칩셋과 유체 고리가 독립적으로 멀티 캐스팅됩니다.

3. 화면 폭 최적화 동적 옥타브

제한된 모바일 해상도 화면을 고려하여, 하단 스위처를 통해 선택된 활성 옥타브 영역의 주파수와 데이터셋만 캔버스에 실시간 리바인딩하는 동적 뷰 변환 아키텍처를 구현했습니다.
